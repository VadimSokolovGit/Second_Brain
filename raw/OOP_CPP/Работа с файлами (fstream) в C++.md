# Работа с файлами (fstream) в C++

> [!info] 🎯 Определение
> Для работы с файлами в C++ используется библиотека `<fstream>`. Она предоставляет классы файловых потоков, унаследованных от базовых потоков ввода-вывода (они работают аналогично `std::cin` / `std::cout`).

## 🗂️ Виды файловых потоков

1. **`std::ofstream`** (Output File Stream): Поток для *записи* данных в файл. Если файла нет — он будет создан. Если файл существовал — его содержимое полностью затирается по умолчанию.
2. **`std::ifstream`** (Input File Stream): Поток для *чтения* данных из файла. Если файла физически нет, поток перейдет в состояние ошибки.
3. **`std::fstream`**: Универсальный двунаправленный поток, позволяющий одновременно и читать, и писать в один и тот же файл.

## 🛠 Режимы открытия файлов (File Modes)

При открытии файла (в конструкторе или методе `.open()`) вторым аргументом можно передать флаги (режимы). Их можно комбинировать через побитовое ИЛИ `|`:

- `std::ios::in` — открыть для чтения (дефолтный для `ifstream`).
- `std::ios::out` — открыть для записи (дефолтный для `ofstream`).
- `std::ios::app` (Append) — запись производится *строго в конец* файла, не затирая старые данные.
- `std::ios::binary` — открыть файл в *бинарном* режиме (символы `\n` записываются как есть, без автоматической трансформации в `CRLF` для Windows).

## ⚙️ Алгоритм работы с файлом

1. **Создать объект потока.**
2. **Открыть файл** (через конструктор или метод `.open()`).
3. **Проверить открытие**: обязательно вызывать метод `.is_open()`. Если у программы нет прав доступа или файла не существует (для чтения), метод вернет `false`.
4. **Провести чтение или запись** (через операторы `<<` / `>>` или методы вроде `std::getline`).
5. **Закрыть файл** методом `.close()`. *(Хотя деструктор закрывает файлы при выходе из зоны видимости, закрывать вручную — признак хорошего тона).*

## 💻 Пример использования

```cpp
#include <iostream>
#include <fstream> 
#include <string>

int main() {
    setlocale(LC_ALL, "Russian");
    std::string filename = "exam_data.txt";

    // ==========================================
    // 1. ЗАПИСЬ В ФАЙЛ (std::ofstream)
    // ==========================================
    std::ofstream outFile(filename); 
    
    if (!outFile.is_open()) {
        std::cerr << "Не удалось открыть файл для записи!\n";
        return 1;
    }

    outFile << "Студент: Иван Иванов\n";
    outFile << "Вопрос по C++: №42\n";
    outFile << "Оценка: Отлично\n";

    outFile.close(); 
    std::cout << "Данные успешно записаны.\n\n";

    // ==========================================
    // 2. ЧТЕНИЕ ИЗ ФАЙЛА (std::ifstream)
    // ==========================================
    std::ifstream inFile(filename); 
    
    if (!inFile.is_open()) {
        std::cerr << "Не удалось открыть файл для чтения!\n";
        return 1;
    }

    std::string line;
    // std::getline считывает файл построчно, пока он не закончится
    while (std::getline(inFile, line)) { 
        std::cout << line << "\n";
    }

    inFile.close(); 
    return 0;
}
```
