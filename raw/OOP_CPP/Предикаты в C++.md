# Предикаты в C++

При работе со стандартными алгоритмами C++ (поиск, сортировка, фильтрация) часто требуется передавать пользовательские критерии. Для этого используются предикаты.

> [!info] Что такое предикат?
> **Предикат** — это любая вызываемая (callable) сущность (обычная функция, указатель на функцию, лямбда-выражение или объект-функтор), которая принимает один или несколько аргументов и обязательно возвращает логическое значение `bool` (`true` или `false`).
> 
> Алгоритмы STL вызывают переданный им предикат для каждого элемента контейнера, чтобы понять, подпадает ли элемент под заданное условие.

## Виды предикатов

В зависимости от количества принимаемых аргументов предикаты делятся на два типа:

1. **Унарные предикаты**
   - Принимают ровно **один** аргумент.
   - *Назначение:* Проверка свойства одного элемента (например, "четное ли число?", "начинается ли строка с буквы А?").
   - *Где используется:* В алгоритмах фильтрации, поиска и подсчета (например, `std::count_if`, `std::find_if`, `std::remove_if`).

2. **Бинарные предикаты**
   - Принимают ровно **два** аргумента.
   - *Назначение:* Сравнение двух элементов между собой (например, "первое число больше второго?").
   - *Где используется:* В алгоритмах сортировки и поиска экстремумов (например, `std::sort`, `std::max_element`).

## Пример использования (Сортировка и Подсчет)

> [!example] Код
> ```cpp
> #include <iostream>
> #include <vector>
> #include <algorithm> // Обязательно для алгоритмов STL
> 
> // 1. УНАРНЫЙ ПРЕДИКАТ (Обычная функция)
> // Проверяет, является ли число отрицательным
> bool isNegative(int x) {
>     return x < 0;
> }
> 
> // 2. БИНАРНЫЙ ПРЕДИКАТ (Обычная функция)
> // Сравнивает два числа. Возвращает true, если первое число БОЛЬШЕ второго.
> // Это заставит std::sort отсортировать массив по убыванию.
> bool compareDescending(int a, int b) {
>     return a > b;
> }
> 
> int main() {
>     std::vector<int> numbers = { 5, -3, 12, 0, -8, 4, -1 };
> 
>     // --- Тест унарного предиката ---
>     // std::count_if считает элементы, для которых isNegative вернет true
>     long long negativeCount = std::count_if(numbers.begin(), numbers.end(), isNegative);
>     std::cout << "Количество отрицательных: " << negativeCount << "\n"; // Выведет 3
> 
>     // --- Тест бинарного предиката ---
>     // Передаем наш бинарный предикат третьим аргументом в сортировку
>     std::sort(numbers.begin(), numbers.end(), compareDescending);
> 
>     std::cout << "После сортировки по убыванию: ";
>     for (int n : numbers) std::cout << n << " ";
>     // Выведет: 12 5 4 0 -1 -3 -8
> 
>     return 0;
> }
> ```
