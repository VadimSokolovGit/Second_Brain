# Контейнер std::list в C++

> [!info] Теоретическая база
> В отличие от `std::vector` (который является массивом и хранит элементы строго друг за другом в едином блоке памяти), **`std::list` — это двусвязный список (Doubly Linked List)**.
> 
> В памяти элементы списка разбросаны по куче хаотично. Каждый элемент упакован в узел (Node), который хранит:
> 1. Сами данные.
> 2. Указатель на следующий узел (`next`).
> 3. Указатель на предыдущий узел (`prev`).

## Плюсы и минусы (Сравнение с std::vector)

> [!success] Плюсы
> - **Мгновенная вставка и удаление $O(1)$**: Вставка в середину или начало не требует сдвига остальных элементов (как в векторе). Происходит просто "перекидывание" указателей соседних узлов.

> [!warning] Минусы
> - **Нет произвольного доступа $O(n)$**: Отсутствует оператор квадратных скобок (нельзя написать `my_list[5]`). Чтобы найти элемент, нужно шагать по указателям от начала.
> - **Больший расход памяти**: Хранение двух дополнительных указателей на каждый элемент (дополнительные 16 байт на узел в 64-битных системах).

> [!important] Главный вывод для экзамена
> Используйте `std::vector`, если нужен частый доступ по индексу и добавление идёт в основном в конец. 
> Используйте `std::list`, если нужно постоянно вставлять или удалять элементы из середины или начала.

## Основные методы

Поскольку список двусвязный, он одинаково эффективно работает с обоими концами:
- `push_back(val)` / `push_front(val)` — добавить в конец / в начало.
- `pop_back()` / `pop_front()` — удалить с конца / с начала.
- `unique()` — удалить последовательно идущие дубликаты.
- `sort()` — встроенная сортировка (алгоритм `std::sort` не работает, так как требует произвольного доступа).

## Пример использования

> [!example] Код
> ```cpp
> #include <iostream>
> #include <list> // Обязательный заголовочный файл
> 
> int main() {
>     setlocale(LC_ALL, "Russian");
> 
>     // Создаем список целых чисел
>     std::list<int> myList;
> 
>     // 1. Добавляем элементы в разные концы списка
>     myList.push_back(20);  // Список: 20
>     myList.push_back(30);  // Список: 20 -> 30
>     myList.push_front(10); // Список: 10 -> 20 -> 30 (вставили в начало)
> 
>     std::cout << "--- Исходный список ---\n";
>     // Обходим список можно через range-based for
>     for (int n : myList) {
>         std::cout << n << " "; // Выведет: 10 20 30
>     }
>     std::cout << "\n\n";
> 
>     // 2. Доступ к элементам по индексу ЗАПРЕЩЕН:
>     // int x = myList[1]; // ОШИБКА КОМПИЛЯЦИИ! Оператор [] не поддерживается.
> 
>     // 3. Работа со специфичными методами списка
>     myList.push_back(20); // Добавим дубликат в конец. Список: 10, 20, 30, 20
>     myList.sort();        // Сортируем список собственным методом. Станет: 10, 20, 20, 30
>     myList.unique();      // Удаляет идущие подряд дубликаты. Станет: 10, 20, 30
> 
>     std::cout << "--- Список после сортировки и удаления дубликатов ---\n";
>     for (int n : myList) {
>         std::cout << n << " "; // Выведет: 10 20 30
>     }
>     std::cout << "\n";
> 
>     return 0;
> }
> ```
