# 📚 Действия над комплексными числами

[[00 Оглавление - Комплексные числа]]

> [!info] Сложение
> **Сложение** и вычитание

*   > [!info] Сложение
  > Сумма двух чисел $z_1 = x_1 + iy_1$ и $z_2 = x_2 + iy_2$ определяется равенством $z_1 + z_2 = (x_1 + x_2) + i(y_1 + y_2)$. Геометрически комплексные числа складываются по правилам векторного сложения. Для сложения справедливо неравенство треугольника: $|z_1 + z_2| \le |z_1| + |z_2|$.

*   > [!info] Вычитание
  > Действие, обратное сложению, определяемое как $z_1 - z_2 = (x_1 - x_2) + i(y_1 - y_2)$. Геометрически модуль разности $|z_1 - z_2| = \sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2}$ равен расстоянию $d$ между точками, изображающими эти числа на плоскости.

> [!info] Умножение
> **Умножение**

*   В алгебраической форме: $z_1 z_2 = (x_1 x_2 - y_1 y_2) + i(x_1 y_2 + y_1 x_2)$ (раскрытие скобок с учетом того, что $i^2 = -1$).

*   В тригонометрической форме: модули перемножаются, а аргументы складываются, т.е. $z_1 z_2 = r_1 r_2 (\cos(\varphi_1 + \varphi_2) + i \sin(\varphi_1 + \varphi_2))$.

*   > [!info] формула Муавра
  > Для возведения в $n$-ю степень применяется **формула Муавра**$z^n = r^n(\cos n\varphi + i \sin n\varphi)$.

> [!info] Деление
> **Деление**
> Деление — это действие, обратное умножению ($z_1 / z_2 = z$). 

*   На практике, для деления в алгебраической форме, числитель и знаменатель дроби умножают на число, сопряженное знаменателю («избавляются от мнимости в знаменателе»). 

*   В тригонометрической форме при делении чисел их модули делятся, а аргументы вычитаются: $\frac{z_1}{z_2} = \frac{r_1}{r_2}(\cos(\varphi_1 - \varphi_2) + i \sin(\varphi_1 - \varphi_2))$.

> [!info] Извлечение корней
> **Извлечение корней**
> Корнем $n$-й степени из комплексного числа $z$ называется такое число $w$, которое удовлетворяет равенству $w^n = z$.
> Извлечение корня из числа $z \neq 0$ дает ровно $n$ различных значений, которые вычисляются по формуле:
> $$w_k = \sqrt[n]{r} \left( \cos \frac{\varphi + 2\pi k}{n} + i \sin \frac{\varphi + 2\pi k}{n} \right)$$
> где $k = 0, 1, 2, \dots, n-1$, а $\sqrt[n]{r}$ — арифметический корень из модуля.
