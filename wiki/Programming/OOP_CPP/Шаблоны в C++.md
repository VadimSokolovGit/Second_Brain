# Шаблоны в C++ (Templates)

> [!info] Что такое шаблоны
> **Шаблоны в C++** — это инструмент обобщенного программирования (*Generic Programming*). Они позволяют написать код (класс или функцию) один раз так, чтобы он работал с абсолютно любыми типами данных (`int`, `double`, `std::string` или пользовательскими классами).

> [!example] Аналогия
> Представьте себе пластиковую формочку для выпечки в виде звезды. Вы можете залить туда тесто и получить печенье-звезду. Можете залить гипс и получить гипсовую звезду. Форма (алгоритм) одна и та же, а материал (тип данных) — разный.

> [!note] Особенности компиляции
> Когда вы пишете шаблонный класс, компилятор не создает реальный код сразу. Он ждет, пока вы вызовете этот шаблон в коде (например, `Array<int>`), и в этот момент **инстанцирует** (генерирует) нужную копию класса под конкретный тип на этапе компиляции (*Compile-time*). Поэтому шаблоны не несут потерь в скорости работы программы (в рантайме).

## Специализация шаблонов (Template Specialization)

Иногда общий алгоритм шаблона отлично работает для 99% типов данных, но для одного конкретного типа он либо неэффективен, либо логически неверен. В таких случаях применяют специализацию.

> [!abstract] Виды специализации
> 1. **Полная (явная) специализация:** Когда класс полностью переписывается для одного конкретного типа (например, строго для `bool`).
> 2. **Частичная специализация:** Когда конкретизируется только часть параметров шаблона (например, для всех типов-указателей `T*`). *Важно:* частичная специализация разрешена только для классов; для функций она запрещена.

> [!example] Зачем это нужно: Классический пример с `std::vector<bool>`
> Обычный `bool` занимает в памяти 1 байт (8 бит), хотя для хранения `true/false` нужен всего 1 бит.
> Разработчики стандартной библиотеки написали **полную специализацию** для `std::vector<bool>`, которая упаковывает каждые 8 логических переменных в 1 байт памяти с помощью побитовых сдвигов. Это экономит память в 8 раз!

## Пример кода: Шаблон и его специализация

> [!example] Код
> ```cpp
> #include <iostream>
> 
> // 1. СНАЧАЛА: Общий шаблон (Базовый чертеж)
> template <typename T>
> class Printer {
> private:
>     T value; // Поле обобщенного типа T
> 
> public:
>     Printer(T val) : value(val) {}
> 
>     void print() const {
>         std::cout << "[Общий шаблон]: " << value << "\n";
>     }
> };
> 
> // 2. ЗАТЕМ: Специализация (Исключение из правил для типа bool)
> template <>
> class Printer<bool> {
> private:
>     bool value; // Для bool мы используем конкретный тип
> 
> public:
>     Printer(bool val) : value(val) {}
> 
>     // Переопределенная логика: выводим текст вместо цифр 1 и 0
>     void print() const {
>         std::cout << "[Специализация для bool]: " << (value ? "YES" : "NO") << "\n";
>     }
> };
> 
> // 3. В КОНЦЕ: Использование в коде
> int main() {
>     setlocale(LC_ALL, "Russian");
> 
>     // Компилятор подставит T = double
>     Printer<double> p1(3.14); 
>     p1.print(); // Выведет: [Общий шаблон]: 3.14
> 
>     // Компилятор перехватит этот вызов и возьмет специализацию
>     Printer<bool> p2(true);   
>     p2.print(); // Выведет: [Специализация для bool]: YES
> 
>     // Пример работы общего шаблона (T = int)
>     Printer<int> p3(42);
>     p3.print(); // Выведет: [Общий шаблон]: 42
> 
>     return 0;
> }
> ```
> 

## Пример кода: Шаблон функции и класса

> [!example] Использование шаблона функции и шаблона класса вместе
> ```cpp
> #include <iostream>
> #include <string>
> 
> // ==========================================
> // 1. ШАБЛОННАЯ ФУНКЦИЯ
> // ==========================================
> // Принимает две переменные по ссылке и меняет их местами
> template <typename T>
> void swapData(T& a, T& b) {
>     T temp = a;
>     a = b;
>     b = temp;
> }
> 
> // ==========================================
> // 2. ШАБЛОННЫЙ КЛАСС
> // ==========================================
> // Хранит пару элементов одинакового типа T
> template <typename T>
> class Pair {
> private:
>     T first;
>     T second;
> 
> public:
>     Pair(T f, T s) : first(f), second(s) {}
> 
>     void print() const {
>         std::cout << "Пара: {" << first << ", " << second << "}\n";
>     }
> 
>     // Метод класса использует нашу внешнюю шаблонную функцию!
>     void invert() {
>         swapData(first, second); 
>     }
> };
> 
> int main() {
>     setlocale(LC_ALL, "Russian");
> 
>     std::cout << "--- Тест шаблонной функции ---\n";
>     int x = 10, y = 20;
>     // Компилятор САМ видит, что x и y — это int, и вызывает swapData<int>
>     swapData(x, y); 
>     std::cout << "x = " << x << ", y = " << y << "\n"; // Выведет: x = 20, y = 10
> 
> 
>     std::cout << "\n--- Тест шаблонного класса ---\n";
>     // Явно указываем тип <double> для класса
>     Pair<double> myDoublePair(1.5, 9.9);
>     myDoublePair.print();  // Выведет: Пара: {1.5, 9.9}
>     
>     myDoublePair.invert(); // Меняем местами внутренние double
>     myDoublePair.print();  // Выведет: Пара: {9.9, 1.5}
> 
> 
>     // Тот же чертеж класса, но теперь для строк <std::string>
>     Pair<std::string> myStringPair("Hello", "World");
>     myStringPair.print();  // Выведет: Пара: {Hello, World}
> 
>     return 0;
> }
> ```
