# Функторы (Объекты-функции) в C++

**Функтор** (или *объект-функция*, *Function Object*) — это обычный класс или структура, в которой перегружен оператор вызова функции — круглые скобки `operator()`. 

Благодаря этой перегрузке мы можем обращаться с объектом класса так, будто он сам является функцией.

## Зачем нужны функторы, если есть обычные функции?
(Частый вопрос на экзамене по STL)

У функторов есть два критических преимущества перед обычными указателями на функции:

> [!success] 1. Сохранение состояния (State)
> Обычная функция «глупа»: она выполнилась и забыла все промежуточные данные (если не использовать опасные глобальные переменные). 
> Функтор — это полноценный объект. У него могут быть `private` поля данных. Мы можем настроить функтор через его конструктор, и он будет *помнить* эти настройки между вызовами.

> [!success] 2. Производительность (Inlining)
> Когда мы передаем в алгоритм (например, `std::sort`) обычную функцию, мы передаем указатель. Компилятору тяжело оптимизировать такой код. 
> Передавая функтор, мы передаем объект конкретного типа. Компилятор видит точный код `operator()` и может «вшить» (заинлайнить) его тело прямо внутрь цикла алгоритма, что работает значительно быстрее.

## Пример работы функтора

В примере ниже создается функтор `Adder`, который хранит число в своем внутреннем состоянии и прибавляет его к передаваемым аргументам, а затем используется в алгоритме `std::transform`.

> [!example] Код
> ```cpp
> #include <iostream>
> #include <vector>
> #include <algorithm>
> 
> // Объявляем класс-функтор
> class Adder {
> private:
>     int valueToAppend; // Внутреннее состояние функтора
> 
> public:
>     // Конструктор настраивает состояние
>     Adder(int x) : valueToAppend(x) {}
> 
>     // Перегружаем оператор ()
>     int operator()(int currentNumber) const {
>         return currentNumber + valueToAppend;
>     }
> };
> 
> int main() {
>     setlocale(LC_ALL, "Russian");
> 
>     // --- 1. Прямой вызов объекта как функции ---
>     Adder addTen(10); // Создаем функтор со значением 10 внутри
>     int result = addTen(5); // Вызываем объект addTen как функцию!
>     std::cout << "Результат addTen(5): " << result << "\n"; // Выведет 15
> 
>     // --- 2. Использование функтора в алгоритмах STL ---
>     std::vector<int> numbers = { 1, 2, 3, 4, 5 };
>     std::vector<int> output(5); // Сюда запишем результат
> 
>     // Создаем функтор "на лету" и передаем его в std::transform
>     std::transform(numbers.begin(), numbers.end(), output.begin(), Adder(100));
> 
>     std::cout << "После функтора Adder(100): ";
>     for (int n : output) std::cout << n << " ";
>     // Выведет: 101 102 103 104 105
> 
>     return 0;
> }
> ```
