# Оператор присваивания копированием в C++

> [!info] Определение
> **Оператор присваивания копированием** (`operator=`) вызывается, когда необходимо скопировать данные из одного **уже существующего** объекта в другой **уже существующий** объект.

## Отличие от конструктора копирования

- **Конструктор копирования** (`Cat cat2 = cat1;`) — выделяет новую память и создаёт объект *с нуля*.
- **Оператор присваивания** (`cat2 = cat1;`) — работает с объектами, которые *уже созданы* и могут содержать старые данные, требующие предварительной очистки.

## Правила правильной реализации

> [!warning] 3 обязательных этапа реализации
> При написании оператора присваивания копированием необходимо строго соблюдать следующий алгоритм:
> 
> 1. **Проверка на самоприсваивание** (`if (this == &other)`)
>    Защищает от ситуаций вида `obj = obj;`. Без этой проверки можно случайно удалить собственные данные объекта перед их копированием.
> 2. **Освобождение старых ресурсов**
>    Очистка памяти (`delete[] data;`), которая ранее принадлежала левому объекту, чтобы избежать утечки памяти.
> 3. **Копирование данных и возврат ссылки**
>    Выделение новой памяти, копирование данных из правого объекта и возврат ссылки на текущий объект (`return *this;`), что позволяет делать цепочки присваиваний (например: `a = b = c;`).

> [!example] Пример реализации
> ```cpp
> class MyString {
> private:
>     char* data;
> public:
>     // ОПЕРАТОР ПРИСВАИВАНИЯ КОПИРОВАНИЕМ
>     MyString& operator=(const MyString& other) {
>         // 1. Защита от самоприсваивания
>         if (this == &other) {
>             return *this; 
>         }
>         
>         // 2. Очистка старых данных
>         delete[] data;
>         
>         // 3. Выделение памяти и копирование
>         data = new char[strlen(other.data) + 1];
>         strcpy(data, other.data);
>         
>         // 4. Возврат ссылки на себя
>         return *this;
>     }
> };
> ```
