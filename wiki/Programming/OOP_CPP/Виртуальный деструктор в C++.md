# Виртуальный деструктор в C++

> [!warning] Основная проблема
> При удалении объекта производного класса (например, `Derived`) через указатель на базовый класс (`Base*`), компилятор по умолчанию вызывает **только деструктор базового класса**.
> 
> **Последствия:** Это приводит к **утечке памяти** и ресурсов, которые были специфичны для производного класса, так как его деструктор не будет вызван.

## Решение

Необходимо объявлять деструктор базового класса с ключевым словом `virtual`. Это гарантирует, что при удалении объекта через базовый указатель будут вызваны деструкторы всей цепочки наследования, начиная с самого производного класса.

> [!important] Золотое правило C++
> Если класс подразумевает, что от него будут наследоваться (в нём есть хотя бы одна виртуальная функция), его деструктор **ОБЯЗАН** быть виртуальным. 
> *Даже если он абсолютно пустой: `virtual ~Base() = default;`*

## Механизм связывания (Под капотом)

Для понимания механизма нужно различать два типа:
- **Статический тип** — тип указателя, известен на этапе компиляции (например, `Base*`).
- **Динамический тип** — тип реального объекта в памяти (в Runtime, например, `Derived`).

> [!info] Раннее связывание (без `virtual`)
> Используется **статический полиморфизм**. Компилятор смотрит только на статический тип указателя. Вызывается деструктор базового класса, а о наследнике компилятор на этапе компиляции не знает.

> [!success] Позднее связывание (с `virtual`)
> Включается **динамический полиморфизм**. Для класса создаётся таблица виртуальных методов — **VMT** (*Virtual Method Table*).
> В Runtime программа обращается к VMT реального объекта, находит деструктор динамического типа (`~Derived()`), выполняет его (очищая ресурсы), после чего автоматически по цепочке вызывается деструктор базового класса.

> [!example] Пример утечки без виртуального деструктора
> 
> ```cpp
> class Base {
>     int id;
> public:
>     ~Base() {} // НЕ виртуальный!
> };
> 
> class Derived : public Base {
>     int* large_buffer; // Указатель на огромный кусок памяти
> public:
>     Derived() { large_buffer = new int[1000000]; } // Выделили ~4 Мб
>     ~Derived() { delete[] large_buffer; } 
> };
> 
> int main() {
>     Base* ptr = new Derived();
>     delete ptr; // Вызовется ТОЛЬКО ~Base(). 
>                 // Около 4 Мегабайт памяти безвозвратно утекли!
> }
> ```
